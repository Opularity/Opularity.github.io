Live USBs.

A live USB is a USB flash drive or a USB external hard disk drive containing a full operating system that can be booted. Live USBs are closely related to live CDs, but sometimes have the ability to persistently save settings and permanently install software packages back onto the USB device. - Wikipedia

Whoa, what? Whats that about?

Basically, a live USB is an entire operating system installed on a USB that you can boot into on most computers. A simple way to do this is to download an iso, burn it onto a usb and boot into it.

What?

  1. A popular "flavor", or type of operating system is Linux. The special thing about Linux is that its "Open Source" (Read more, here) meaning its free for people to write and manipulate (for good, or bad). A popular "flavor" of Linux is called Ubuntu, and its what we'll be using in this demonstration.

    Download it here.

  2. Once downloaded, install an iso (the file type of the Ubuntu file) writer that will put the iso file on your usb called UNetbootin.

    Download it here.

  3. Locate your iso file with the "..." button.
  4. Plug in your USB, then select your USB Drive in the "Drive section"
  5. When everything is working, click OK, and let it install.
  6. Now heres the trickier bit: Booting into it.
    Once you've installed the iso onto the USB with the UNetbootin, you need to shut down your computer, and when its turning on again, you need to press something like F12 (if its a Dell computer, or a native Windows computer), if its a Mac, you may be screwed (try holding option/alt on startup).

Once done, you should have a menu like this

Hit "Try ubuntu without installing", and you're done!
